## 🛠️ Project — Password Generator

A command-line password generator that asks the user how many **letters**, **symbols**, and **numbers** they want, then builds a randomized, shuffled password.

### How it works

1. Ask the user for the number of letters, symbols, and numbers
2. Use `for` loops to randomly pick characters from each list
3. Combine everything into one list
4. Shuffle with `random.shuffle()` so the characters are in a random order
5. Join and print the final password


No external libraries needed — only the built-in `random` module.
